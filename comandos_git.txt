Comandos Essenciais:
git init    inicializa um novo repositório Git em um diretório existente. 
git add     adiciona arquivos ou alterações ao "staging area" (área de preparação) para serem incluídos no próximo commit. 
git commit  cria um novo commit com as alterações preparadas na área de staging, adicionando-as ao histórico do repositório. 
git status  mostra o estado atual dos arquivos no seu diretório de trabalho, indicando quais arquivos foram modificados, adicionados ou estão prontos para commit. 
git         log mostra todo o histórico de commits
